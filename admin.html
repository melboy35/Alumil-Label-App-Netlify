Azure Blob Storage Setup

  1. 1. Upload your Excel/CSV/JSON file to Azure Blob Storage
  2. 2. Generate a SAS URL with Read permissions
  3. 3. Ensure CORS is configured to allow your domain
  4. 4. Use the SAS URL in the "Connect Azure Blob Storage" button

Supported formats: .xlsx, .xls, .csv, .json
